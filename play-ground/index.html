This is the demo for tiny-lang

Helper functions

  • Print([args]) Ex: print("hello", "world") -> console.log("hello", "world")
  • Add([args]) Example: add(1, 2, 3) -> 6
  • Multiply([args]) Example: multiply(2, 2, 2) -> 8
  • Subtract([args]) Example: subtract(5, 3, 1) -> 1
  • Divide([args]) Example: divide(9, 3, 2) -> 1.5
